# Menu API

Menu API server using `graphql`.

# Getting Started

To run locally:

1. Install `node v20`.
2. Install `yarn v4`.

To run with Docker CLI:

1. Run `docker/build`.
2. Run `docker/launch`.

To run with VSCode Dev Containers:

1. Run `docker/build`.
2. In VSCode, run `Reopen in Container`.
